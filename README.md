# CSV Task Manager

Simple task and goal manager that stores data in CSV

# Usage

```sh
# Clone this repo
git clone https://github.com/maybe-danny/csv-tasks
cd csv-tasks

# Run the program using dotnet and specify the path to the CSV document
# If the file does not exist, it will be created in the process
dotnet run tasks.csv
```
